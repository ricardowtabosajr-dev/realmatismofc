# TeamManager - Gestor de Times de Futebol ⚽

O TeamManager é uma aplicação web moderna projetada para ajudar treinadores e organizadores de times de futebol amador a gerenciar seus elencos, agendar jogos e organizar as convocações de forma eficiente.

## 🚀 Funcionalidades

- **Painel Geral (Dashboard)**: Acompanhe o total de atletas, jogos realizados e arrecadação total do time.
- **Gestão de Atletas**: Cadastro completo com nome, posição e contato de WhatsApp.
- **Gestão de Jogos**: Agendamento de partidas com detalhes de adversário, data, hora, local e taxa de participação.
- **Sistema de Convocação**:
  - Selecione quais atletas participarão de cada jogo.
  - Controle de pagamento (Pago/Pendente) em tempo real.
  - Resumo financeiro por partida.
- **Integração com WhatsApp**: Compartilhe a lista de convocados formatada diretamente no grupo do time.

## 🛠️ Tecnologias Utilizadas

- **React 19**
- **TypeScript**
- **Vite**
- **Lucide React** (Ícones)
- **Vanilla CSS** (Design Premium Dark)

## 📦 Como Rodar o Projeto

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Abra o navegador em `http://localhost:5173`.
